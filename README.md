# Plato's Wallet - Payment Processing + Free Marketing

Premium website for Plato's Wallet payment processing and marketing services. Built with 11ty static site generator.

## Overview

Plato's Wallet offers retail businesses competitive payment processing rates plus **FREE marketing services** funded from processing margins:

- Custom website design ($3,000+ value)
- Local SEO optimization ($1,500/mo value) 
- Google Ads management ($1,000/mo value)
- Social media marketing ($800/mo value)
- No long-term contracts, same-day deposits

## Technology Stack

- **Static Site Generator**: 11ty (Eleventy)
- **CSS**: Custom CSS with CSS Grid and Flexbox
- **JavaScript**: Vanilla JS for interactions
- **Fonts**: Inter (body) + Fraunces (headings)
- **Deployment**: Vercel (auto-deploy from main branch)

## Development

```bash
# Install dependencies
npm install

# Build site
npm run build

# Serve locally
npm run serve
```
